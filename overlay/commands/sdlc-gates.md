# /sdlc-gates —— 双验证门检查

汇总功能门与非功能门。**裁判是确定性工具，不是 AI 自评**——只触发工具、收集结果、汇总绿红，不得主观判定通过。

## 功能门（按金字塔分层）
逐层运行项目实际命令（从 CLAUDE.md / config 读取），收集退出码：单元 → 集成/契约 → API/BDD → E2E。

## 非功能门(NFR)
- 安全：SAST 无高危、依赖无高危漏洞、无密钥、越权负向测试。
- 性能：关键接口压测达基线、无 N+1/全表扫描、复杂度达标。
- 可维护性：重复率/圈复杂度、分层边界。

## 汇总
输出"门 → 命令 → 退出码 → 绿/红"表。
- 两门皆绿 → 可进入发布清单（人工裁决）。
- 任一红 → 失败报告交回执行器受限重试（≤3 次、需有进展、超限升级人工）。
- 禁止为换绿而关闭鉴权/放宽校验/跳过门。
